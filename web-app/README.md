# Lens Designer

A modern React web app for optical lens design, built with Vite, Tailwind CSS, and Lucide React icons.

## Theme

- **Background**: Midnight Blue (#0B1120)
- **Primary**: Electric Cyan (#22D3EE) for buttons and accents
- **Glassmorphism**: Semi-transparent cards with subtle borders
- **Font**: Inter (sans-serif)

## Layout

- **Top**: Navigation bar (Lens Designer, System Editor, Properties, Export)
- **Left**: Main canvas for optical layout drawing
- **Right**: System Properties sidebar

## Setup

```bash
cd web-app
npm install
npm run dev
```

## Build

```bash
npm run build
```
