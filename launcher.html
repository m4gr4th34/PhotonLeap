[ SECURE CONTEXT REQUIRED ]

You opened MacOptics from a file:// location. Modern browsers require a Secure Context (HTTPS or localhost) to run Web Workers—which MacOptics uses for its in-browser Python physics engine (Pyodide).

One-click fix: Run the launcher script for your system. It starts a local server and opens MacOptics in your browser automatically.

  • Windows: Double-click start-windows.bat
  • Mac: Double-click start-mac.command (or right-click → Open)

The script uses Python’s built-in HTTP server—no extra installation needed if Python is already on your system.
